# Employee Schedule Manager

مدير جدول الموظفين - تطبيق ويب تفاعلي لإدارة جداول الموظفين

## 🎯 الميزات

- ✅ جدول تفاعلي لإدارة الموظفين
- ✅ إحصائيات فورية للموظفين
- ✅ رفع ملفات إكسل واستبدال البيانات
- ✅ إضافة وحذف وتعديل الموظفين
- ✅ حفظ البيانات في قاعدة البيانات
- ✅ واجهة احترافية وسهلة الاستخدام

## 📦 المحتويات

- `project-files.zip` - جميع ملفات المشروع
- `README_AR.md` - دليل شامل بالعربية
- `FILES_LIST.md` - قائمة الملفات

## 🚀 البدء السريع

1. استخراج الملفات من `project-files.zip`
2. تثبيت المكتبات: `pnpm install`
3. تشغيل الخادم: `pnpm dev`
